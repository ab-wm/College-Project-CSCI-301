How to Play Sudoku

The main goal of the game is to fill each cell with an appropriate number. There are three core rules that decide whether a value has been correctly placed on the grid. They are:

  • The only copy of the number within the subgrid. The subgrid is the 3*3 grid of cells, for a total of 9 subgrids.
  • The only copy of the number within the row it is placed in.
  • The only copy of the number within the column it is placed in.

How this Program helps, and Does Not Help

The program will automatically check if the placed value in correct in relation with other numbers already placed in the grid, as well as block any attempts to replace numbers that were provided with the puzzle. It is, however, important to note that the program will not check if the value has been correctly placed in relation to the solution of the puzzle. That means it will be possible to end up in a situation where the puzzle will be unsolvable if a mistake has been made earlier.

If you wish to start a new game or quit, you can select the appropriate menu option located on the top right of the window.

Keep in mind that there is only one correct solution to the puzzle.

Be careful and have fun!
